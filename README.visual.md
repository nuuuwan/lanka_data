# Visual Style

1. Consistent color palette with accessible contrast (colorblind-safe, no near-identical hues)
2. Align title/subtitle hierarchy with clear font-size/weight steps, not just size
3. Legend counts formatted consistently (padding, alignment) and ordered by value, not arbitrary
4. Reduce border/gridline weight between hexagons so shapes don't compete with data
5. Add consistent margin/padding around the whole canvas, avoid edge-touching elements
6. Use a muted/neutral background instead of flat grey banner for title
7. Footnote and source text should share one type style, smaller and de-emphasized consistently
8. Add subtle drop shadow or none at all, but be consistent across all elements
9. Ensure label text inside hexagons has enough contrast (dark text on light hues, light text on dark hues) automatically
10. Include a scale/legend note in a consistent location (bottom-left or bottom-right) rather than floating text with different alignment than the title block
